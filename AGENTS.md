# OwnID iOS SDK Agent Guide

Default routing for agents working with the OwnID iOS SDK.

<!-- private-only:start -->
## SDK Development

For SDK development or release work in this internal checkout, use
`.agents/skills/ownid-ios-sdk-development/SKILL.md`. That skill owns the
workflow, source-of-truth order, architecture references, validation selection,
worktree/staging rules, and release/publishing boundaries.

Do not treat SDK internals as a client-app integration surface.
<!-- private-only:end -->

## Client-App Integration

For client-app integration tasks, use
`.agents/skills/ownid-ios-integration/SKILL.md`.

For installation, setup, current requirements, supported dependency channels,
and public integration documentation, use `README.md`, `docs/README.md`, and
the selected public release metadata/tag.
